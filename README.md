# Personalización de Git Bash

Este documento proporciona instrucciones paso a paso para personalizar tu Git Bash para que tenga un prompt específico con colores y que muestre solo el nombre de la carpeta actual y la rama de Git.

## Requisitos

- Git Bash instalado en tu sistema.
- `awk` instalado (generalmente viene preinstalado en Git Bash).

## Instrucciones

### Paso 1: Configurar el archivo `~/.bashrc`

1. **Abrir Git Bash**.
2. **Editar el archivo `~/.bashrc`**:
   ```sh
   nano ~/.bashrc
   ```
3. **Añadir o actualizar el contenido del archivo `~/.bashrc`**:

   ```sh
   # Definir colores
   RED="\[\033[0;31m\]"
   GREEN="\[\033[0;32m\]"
   YELLOW="\[\033[0;33m\]"
   BLUE="\[\033[0;34m\]"
   MAGENTA="\[\033[0;35m\]"
   CYAN="\[\033[0;36m\]"
   BOLD_GREEN="\[\033[1;32m\]"
   RESET="\[\033[0m\]"

   # Git Prompt Customization
   function parse_git_branch {
       git branch 2> /dev/null | grep '^*' | awk '{print $2}'
   }

   PS1="${BOLD_GREEN}→ ${CYAN}\W ${YELLOW}git:(${MAGENTA}\$(parse_git_branch)${YELLOW})${RESET}\$ "
   ```

4. **Guardar y salir** del editor (`Ctrl + X` para salir de `nano`, luego `Y` para confirmar y `Enter` para guardar).

### Paso 2: Configurar el archivo `~/.bash_profile`

1. **Abrir el archivo `~/.bash_profile`**:
   ```sh
   nano ~/.bash_profile
   ```
2. **Añadir o actualizar el contenido del archivo `~/.bash_profile`**:

   ```sh
   # generated by Git for Windows
   test -f ~/.profile && . ~/.profile
   test -f ~/.bashrc && . ~/.bashrc

   if [ -f ~/.bashrc ]; then
       source ~/.bashrc
   fi
   ```

3. **Guardar y salir** del editor (`Ctrl + X` para salir de `nano`, luego `Y` para confirmar y `Enter` para guardar).

### Paso 3: Recargar los archivos de configuración

1. **Recargar el archivo `~/.bashrc`**:
   ```sh
   source ~/.bashrc
   ```

2. **Recargar el archivo `~/.bash_profile`**:
   ```sh
   source ~/.bash_profile
   ```

### Verificación

1. **Verificar el contenido de `PS1`**:
   ```sh
   echo $PS1
   ```
   Deberías ver algo como:
   ```
   ${BOLD_GREEN}→ ${CYAN}\W ${YELLOW}git:(${MAGENTA}\$(parse_git_branch)${YELLOW})${RESET}\$
   ```

2. **Abrir una nueva terminal de Git Bash** y navegar a un repositorio Git para verificar que el prompt se muestra correctamente.

### Resultado Final

El prompt debería verse como:

```
→ carpeta git:(rama)$
```

Con los siguientes colores:

- `→` en verde negrita.
- El nombre de la carpeta (`\W`) en cian.
- `git:` en amarillo.
- La rama de Git en magenta.
